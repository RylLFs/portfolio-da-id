Klasifikasi Kualitas Udara Global AQI Menggunakan Data Mining

Mengklasifikasikan kualitas udara global dengan decision tree

Context / Problem

Pada proyek semester lima ini, saya menggunakan Decision Tree Machine Learning pada aplikasi RapidMiner untuk mengklasifikasikan kualitas udara pada beberapa negara dalam data berdasarkan parameter polutan utama seperti PM2.5, PM10, NO₂, SO₂, dan O₃ terhadap standar WHO. Dengan pendekatan berbasis data, insight ini diharapkan dapat menghasilkan informasi terkait Air Quality Index (AQI) yang dapat digunakan untuk edukasi publik dan pengambilan keputusan kebijakan lingkungan yang lebih efektif.

Tech Stack & Methods

Tools

RapidMiner

Algorithm

C4.5 Decision Tree

Classification Type

Multi-Class Classification

Data Mining

Criteria Weighting

Data Source

Kaggle

Dataset Size

3,361 Baris Data

Key Features

Global Air Quality

Analisis komprehensif terhadap PM2.5, PM10, NO₂, SO₂, dan O₃ dengan visualisasi pengaruh masing-masing terhadap AQI.

Decision Tree Model

Penggunaan Algoritma C4.5 Decision Tree model pada RapidMiner dengan akurasi 99%.

WHO Standards

Mengklasifikasikan pembobotan model sesuai dengan standar WHO yang telah ditetapkan.

Insights

Berdasarkan Hasil Analisis, hasil data menunjukkan Kualitas Udara didominasi oleh kualitas Buruk, didomminasi oleh PM2.5 dengan rasio 78.63%.
