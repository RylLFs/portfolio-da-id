Visualisasi dan Analisis Data Gempa di Jawa Barat

Menganalisis pola gempa bumi untuk insight geografis yang bermakna

Context / Problem

Tugas Proyek Akademik pada semester tiga ini, saya mengumpulkan data yang sudah diolah oleh BPS Jawa Barat dan menganalisis 324 data kejadian gempa di pada daerah - daerah di Jawa Barat pada rentang tahun 2012 - 2023 melalui visualisasi dan analisis untuk memberikan insight bermakna.

Tech Stack & Methods

Languages

Python

Libraries

Pandas, NumPy, Matplotlib, Seaborn

Tools

Jupyter Notebook, Visual Studio Code

Methods

Data Analysis, Statistical Analysis

Data Source

BPS Jawa Barat

Dataset Size

324 Baris Data

Key Features

Data Visualization

Visualisasi grafis menggunakan Matplotlib dan Seaborn untuk menampilkan distribusi gempa.

Detailed Analysis

Eksplorasi mendalam dataset untuk menemukan insights dan pola yang tersembunyi.

Geographic Patterns

Analisis pola geografis gempa untuk memahami distribusi dan frekuensi terjadinya di berbagai wilayah Jawa Barat.

Insights

Berdasarkan hasil VIsualisasi, Gempa banyak terjadi di tahun 2017 dengan 284 Kejadian, didominasi oleh wilayah kabupaten Garut sebanyak 70 kejadian, Kabupaten Tasikmalaya sebanyak 61 kejadian, dan Kota Sukabumi sebanyak 58 kejadian.
