Visualisasi dan Analisis Simulasi Gaji Pegawai IT dengan Algoritma Association Rules

Menganalisis pola korelasi gaji menggunakan machine learning

Context / Problem

Tugas Proyek Akademik pada semester empat ini, saya menggunakan algoritma Association Rules pada dataset 400 profil profesional IT yang didapat dari Kaggle. Mengidentifikasi korelasi kuat antara posisi, tahun pengalaman, skill set, dan rentang gaji. Temuan ini dapat membantu gambaran perusahaan dalam menentukan struktur kompensasi yang kompetitif dan membantu profesional IT dalam merencanakan pengembangan karir mereka secara strategis.

Key Features

Association Rules

Implementasi algoritma Association Rules (Apriori) pada dataset 400 profesional IT dengan confidence minimum 70%.

Salary & Experience Categorization

Pembagian data gaji ke dalam kategori range terstruktur (e.g., 40,000-120,000 USD) dan Experience (e.g., Pemula (0-5 tahun)) untuk encoding Association Rules.

Career Factor Correlation Analysis

Analisis mendalam hubungan antara job position, years of experience, technical skills, dan salary range untuk mengidentifikasi besaran nilai gaji karyawan IT.

Insights

Salah satu insght yang didapat adalah Entry-level Web Developers (0-5 tahun) memiliki range Salary range $40K-$120K.
